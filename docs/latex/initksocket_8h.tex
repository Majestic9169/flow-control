\doxysubsection{Detailed Description}
simulates the "{}kernel"{} and services ktp operation calls 

initializes R and S threads to handle receive and send calls.

Initializes shared memory for details regarding SOCK\+\_\+\+KTP sockets.

Creates a garbage collector thread to clean up SOCK\+\_\+\+KTP sockets when their parent process dies
