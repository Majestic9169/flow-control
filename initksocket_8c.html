◆ recv_routine()

void * recv_routine ( void *  args)

receiver thread routine

the receiver thread routine

waits for a message in a recvfrom() call from any of the UDP socks using select(). Data packets are written to recv_buf or held in rwnd.ooo (out-of-order, so no ACK). On select() timeout, sends dup ACK if nospace has since cleared.

sender thread routine

the sender thread routine

sleeps for time T/2 and wakes up periodically. Retransmits swnd if timeout T is exceeded. Sends any pending messages from send_buf that fit within the current swnd and remote rwnd.
